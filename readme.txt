== Description ==

Ce plugin permet de pouvoir poster un article ou une page wordpress en même temps sur le site de Tipao en quelques clics.
Afin de pouvoir poster également votre article sur Tipao, vous devez renseigner les identifiants de connexion de votre Compte Tipao (Identifiant ou adresse mail, mot de passe) dans la page d'options et mettre à jour.
Votre ville par défaut sur Tipao devrait alors apparaître, cocher la case 'Postez automatiquement?' vous permettra de ne pas cocher la case 'Poster l'article sur tipao?' lors de la création d'un article/page.
Il ne vous restera plus qu'à configurer votre article Tipao en choisissant une catégorie ainsi que la ville qui peut être modifiée (en tapant le nom de la ville ou en choisissant dans l'historique des villes).
Enfin avant de publier, vérifiez que la case soit bien coché, un message vous confirmera que l'article est bien posté sur wordpress ainsi que sur Tipao.

== Installation ==

Wordpress :  Telecharger le dossier `Tipao` et activer le Plugin grâce au menu Plugin de Wordpress.

Sinon extrayez les fichiers du .zip et placer le contenu dans le répertoire wp-content/plugins/de votre installation WordPress et activez ensuite le Plugin grâce à la page de Plugins Wordpress.

Vous devriez trouver le menu 'Tipao' dans votre panneau d'administration wordpress ainsi qu'un lien vous demandant de configurer vos paramètres.

Si vous mettez à jour une ancienne version, utilisez s'il vous plaît le bouton 'Réinitialisez les Options'.

== Questions? ==

= Est-ce que le Plugin ajoute des tables dans la base de données? =

Non, la liaison et les données associées sont sauvées dans la table options de wordpress.

= Un problème avec le Plugin?  =

Vous pouvez soumettre une demande d'assistance sur http://tipao.com.
